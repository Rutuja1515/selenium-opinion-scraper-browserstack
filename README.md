# selenium-opinion-scraper-browserstack

# Overview

This project is a Selenium-based automation script that scrapes articles from the **El País Opinion section**.
The script extracts article titles and content in Spanish, downloads article images, translates titles into English, and performs basic text analysis on the translated titles.

# Features

 Navigate to the **El País Opinion page**
 Scrape the **first five articles**
 Print **article titles and content in Spanish**
 Download **cover images** of each article
 Translate **article titles from Spanish to English**
 Perform **word frequency analysis** on translated titles
 Demonstrate **cross-browser testing using BrowserStack**

# Technologies Used

 Python
 Selenium WebDriver
 WebDriver Manager
 Requests Library
 Google Translate API (googletrans)
 BrowserStack (for cross-browser testing)


# Output

The script will:

 Print article titles and content in the console
 Display translated article headers
 Show repeated words appearing more than twice
 Download article images into the **images/** folder

# Cross-Browser Testing

The script includes a demonstration of **BrowserStack cross-browser testing** using multiple browsers and threads.
